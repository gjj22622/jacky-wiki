# 軟體清單（GUI 應用）

> 需要手動安裝的桌面應用程式。換機時一個個裝回來。
> CLI 命令列工具另見 [CLI工具清單](CLI工具清單.md)。

---

## 必裝（缺了無法工作）

| 軟體 | 用途 | 官方下載 | Windows 一鍵安裝 | 驗證 |
|---|---|---|---|---|
| **Claude Code** | 主 IDE / AI 對話（整套作業系統的核心）| https://claude.ai/code | 依官網指示 | `claude --version` |
| **Git** | 版本控制 / clone wiki | https://git-scm.com/download/win | `winget install Git.Git` | `git --version` |
| **Node.js（LTS）** | Claude Code / 自建 script 的執行環境 | https://nodejs.org | `winget install OpenJS.NodeJS.LTS` | `node -v`（應 ≥ v24）|
| **Docker Desktop** | 跑 AI 大腦倉庫（AnythingLLM）等容器 | https://docker.com/products/docker-desktop | `winget install Docker.DockerDesktop` | `docker --version` |

---

## 常用（沒有會不方便）

| 軟體 | 用途 | 官方下載 | Windows 一鍵安裝 |
|---|---|---|---|
| **VS Code** | 文字編輯 / 看 diff | https://code.visualstudio.com | `winget install Microsoft.VisualStudioCode` |
| **GitHub CLI（gh）**| 命令列操作 GitHub（PR/issue）| https://cli.github.com | `winget install GitHub.cli` |
| **OneDrive** | 素材中繼站（草案區、ailab inbox）| Windows 內建 | （內建）|
| **Python 3.12** | 部分 skill（問卷分析/tts/docx）需要 | https://python.org/downloads | `winget install Python.Python.3.12` |
| **Obsidian** | 用圖形介面瀏覽 wiki（可選）| https://obsidian.md | `winget install Obsidian.Obsidian` |

## 安裝順序建議

1. **Git** → 才能 clone wiki
2. **Node.js** → Claude Code 依賴
3. **Claude Code** → 整套作業系統入口
4. **Docker Desktop** → 要跑 AI 大腦才需要（可延後）
5. 其餘按需安裝

> 完整逐步指令見 [換電腦SOP](../migration/換電腦SOP.md)。
