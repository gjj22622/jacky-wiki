# 表單 5｜一頁企劃書（SOSTAC 全收束）

> **對應 SOSTAC**：全 SOSTAC 收束（S+O+S+T+A+C）
> **用途**：將前 4 張表單收斂為**一頁企劃書**——TBSA 進階檢定的最終交付物
> **檢驗標準**：能在一頁內表達完整企劃，是「**真的想清楚**」的試金石
> 來源：`OneDrive/TBSA知識體系/TBSA企劃工具表單欄位說明_2025.pptx` Slide 5

---

## 一頁企劃書架構

```
企劃案名稱：____________

┌──────────────────────────────────────┐
│ 1. 目的與效益（Why / Evaluate）         │
│    ├── 目的／目標                       │
│    └── 效益                            │
├──────────────────────────────────────┤
│ 2. 內容要點（What / Whom）              │
│    ├── 策略目標                         │
│    ├── 關鍵瓶頸／障礙                   │
│    └── 行銷策略（目標市場／客群／定位）   │
├──────────────────────────────────────┤
│ 3. 實施方式（How / Where / When）       │
│    ├── 如何實施（7Ps 戰術辦法）         │
│    ├── 實施期間（一年為限）             │
│    └── 實施範圍（地理區域）             │
├──────────────────────────────────────┤
│ 4. 執行團隊（Who）                      │
│    └── 專案任務編組 + 負責人員名單       │
├──────────────────────────────────────┤
│ 5. 所需經費（How much）                 │
│    └── 分項預算 + 經費總額              │
└──────────────────────────────────────┘
```

---

## 5 個段落詳解

### 1. 目的與效益（Why / Evaluate）

#### 目的／目標

> 先敘述本案的源起與背景，進一步提出本案希望達成的**理想願景、中長期方向目標及短期階段性目標**。

#### 效益

> 根據左列之短期階段性目標進行推估，若能順利達成，將可獲致的**具體效益**——
>
> 例如：業績成長率、獲利成長率、市佔成長率、品牌知名度等及其他重要的利益與成果。

---

### 2. 內容要點（What / Whom）

#### 2.1 策略目標

> 在綜合考量在 SWOT 策略矩陣中的策略議題後，**說明策略矩陣中挑選出的策略作法成為企劃方案最主要的策略議題**。
>
> （來自 [表單 2](表單2_現況分析與策略目標設定.md)）

#### 2.2 關鍵瓶頸／障礙

> 綜合考量本案之策略目標，**從現狀到目標達成過程中所面臨的關鍵瓶頸／障礙**。
>
> （來自 [表單 2](表單2_現況分析與策略目標設定.md) 的關鍵障礙）

#### 2.3 行銷策略

> 根據策略目標，說明所決定的行銷策略要素，包含：
> - **目標市場**
> - **目標客群**
> - **商品定位**
>
> （來自 [表單 3](表單3_STP市場策略設計.md)）

---

### 3. 實施方式（How / Where / When）

> 為能落實解決關鍵性瓶頸障礙，並順利執行行銷策略，以達成階段性目標，依據上述內容要點提到的資訊，說明企劃方案將會：

#### 3.1 如何實施（How）

> 在上述策略議題與行銷策略指導下，**如何運用具體的 7Ps 戰術辦法**（包括創意內容、規劃重點、具體措施等），能有效突破關鍵瓶頸，順利達成目標。
>
> （來自 [表單 4](表單4_行銷戰術及溝通活動設計.md) 的 7Ps）

#### 3.2 實施期間（When）

> 說明本企劃案的實施期間（**以一年為限**）。

#### 3.3 實施範圍（Where）

> 說明各種**通路及推廣戰術的實施地理區域範圍**。

---

### 4. 執行團隊（Who）

> 根據落實各項戰術所需之：
> - **專案任務編組**
> - **負責人員名單**

---

### 5. 所需經費（How much）

> 分項列舉實施各項戰術專案所需之：
> - **分項預算**
> - **經費總額**
>
> （來自 [表單 4](表單4_行銷戰術及溝通活動設計.md) 的行銷溝通預算）

## 5W2H 對應

> 一頁企劃書本質上是 **5W2H** 的整理：

| W/H | 段落 |
|:--:|---|
| **Why** | 1.1 目的／目標 |
| **What** | 2 內容要點（策略 / 障礙 / 行銷策略） |
| **Whom** | 2.3 行銷策略中的目標市場、客群 |
| **How** | 3.1 如何實施（7Ps） |
| **When** | 3.2 實施期間 |
| **Where** | 3.3 實施範圍 |
| **Who** | 4 執行團隊 |
| **How much** | 5 所需經費 |
| **Evaluate** | 1.2 效益（成果目標） |

---

## 為什麼要收斂成一頁

> 鄭沂珊書 §10 強調：
>
> 「**將完整 SOSTAC 六階段濃縮為一頁——檢驗一個企劃案是否真的想清楚的試金石。**」

如果不能用一頁說清楚，代表前 4 張表單**還沒真的想透**。
