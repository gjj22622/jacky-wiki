# AI 作業系統（jos）

> Jacky 的 AI 工作環境 = 一套新世代的作業系統。
> **本域的唯一使命**：**換電腦（或新成員上機）能無痛轉移**——一句指令 `/移機`，照著做就回到原本的工作狀態。

---

## 為什麼有這個域

Jacky 在双云 AI School 提出：**AI 的使用，應該像一套新世代的作業系統。** 既然是作業系統，就牽涉環境、硬體、工具等各種面向。這些東西現在散落在 `~/.claude/`、`bin/`、各種設定檔、甚至腦袋裡——換電腦就得從零重裝、靠回憶，極易漏東漏西。

本域把整套 AI 作業系統**機械化記錄**下來，讓它可複製、可傳承。

---

## 四層架構（這套作業系統長什麼樣）

```
┌─ Skill 層 ─────────────────────────────────┐
│ 16 支個人 skill + 4 個 plugin              │  ← skills/
│ （/wiki /ailab /移機 /作品 /排查 …）        │
└────────────────────────────────────────────┘
            ↑ 跑在
┌─ 工具層（IDE）─────────────────────────────┐
│ Claude Code（主）+ Codex CLI（副）         │  ← environment/CLI工具清單
│ + MCP 伺服器（Drive/Gmail/Calendar…）      │  ← environment/MCP伺服器清單
└────────────────────────────────────────────┘
            ↑ 裝在
┌─ 環境層 ───────────────────────────────────┐
│ 軟體（Docker/Git/VSCode）+ CLI + 環境變數  │  ← environment/
│ + 自建程式（bin/、statusline…）            │
└────────────────────────────────────────────┘
            ↑ 跑在
┌─ 硬體/OS 層 ───────────────────────────────┐
│ Windows 主機 / G2 mini Ubuntu / WSL2       │  ← environment/硬體與作業系統
└────────────────────────────────────────────┘
```

## 🚀 換電腦：從這裡開始

新電腦／新成員上機，**只要做一件事**：

```
打開 Claude Code → 輸入 /移機 restore
```

它會帶你照 [換電腦SOP](migration/換電腦SOP.md) 一步步走。看不懂指令也沒關係，SOP 寫到「複製貼上就行」的程度。

> 不想用指令、想自己照做 → 直接讀 [migration/換電腦SOP.md](migration/換電腦SOP.md)。

## 與 `ailab/tools/` 的分工（重要，避免混淆）

| 域 | 回答的問題 | 性質 |
|---|---|---|
| [ailab/tools/](../ailab/tools/工具棧索引.md) | **為什麼**用這套工具？何時該換、何時淘汰？ | 工具觀（活知識、會辯證） |
| **jos（本域）** | **怎麼**把這套裝回新機？裝了哪些、版本多少、怎麼驗證？ | 操作手冊（機械化清單） |

→ 想理解「Jacky 為什麼這樣選工具」讀 ailab；想「把這套搬到新機」讀 jos。兩邊互相連結，不重複內容。

## 對應 Skill

- **canonical 主版**：`wiki/jos/skill/SKILL.md`（git 管理）
- **各機執行版**：`~/.claude/skills/移機/SKILL.md`（symlink 或 copy）
- **觸發**：`/移機` / `/移機 capture`（更新清單）/ `/移機 restore`（新機還原）/ `/移機 sync`（同步 skill）
- 安裝：[skill/INSTALL.md](skill/INSTALL.md)

---

## 維護節奏

| 時機 | 動作 |
|---|---|
| 裝了新 CLI / MCP / 軟體 | 跑 `/移機 capture` 更新對應清單 |
| 新增/改了 skill | 跑 `/移機 sync` 把 master 回寫到 skills/ |
| 每月健康檢查 | 對照 [還原檢查清單](migration/還原檢查清單.md)，確認清單沒過期 |
