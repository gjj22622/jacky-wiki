# 視覺與文字硬規範

## 字型

| 文字類型 | 唯一允許的預設字型 |
|---|---|
| 中文 | 微軟正黑體（Microsoft JhengHei） |
| 英文 | Arial |
| 數字、百分比、日期、網址、Email | Arial |

- 所有課型、簡報、講義、表格、導覽頁與可視化交付均遵守。
- 中英混排時分 run／span 指定字型；不可只設定整個文字框後假設自動 fallback 正確。
- 不以 Noto Serif TC、Noto Sans TC、Inter、Cormorant Garamond 或其他裝飾字型作為預設。
- 中文粗體使用微軟正黑體 Bold；英文粗體使用 Arial Bold。
- 不用 emoji 代替圖示或段落標籤，避免跨平台渲染成方框。

## 字級

| 用途 | 建議字級 |
|---|---:|
| 封面主標 | 44–60pt |
| 章節標題 | 36–48pt |
| 小標題 | 24–32pt |
| 一般內文 | 20–24pt |
| 表格／卡片內文 | 16–18pt |
| 關鍵數字 | 40–72pt |
| 所有可見文字硬下限 | 16pt |

低於 16pt 直接判定視覺 Gate 不通過。16pt 是最低線，不是一般內文目標。

## 文字

- 演講文字要短、可口說、可記憶；工作坊文字要可操作、可複製、可驗收。
- 每頁只承擔一個主要訊息或任務。
- 避免「賦能、變現、增長飛輪、North Star Metric」等未翻譯黑話。
- 數字優先寫絕對值、期間與比較基準。
- 客戶品牌、學員個資與未授權數據必須脫敏。

## 驗收

1. 掃描所有文字 run，確認中文字型、英數字型及最低字級。
2. 用實際 PowerPoint 或輸出 PDF 渲染，不只檢查程式設定。
3. 以教室後排可讀性判定；若必須縮到 16pt 以下，應刪字、拆頁或改圖，不得硬塞。
