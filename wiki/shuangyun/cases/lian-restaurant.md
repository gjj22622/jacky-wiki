# 濂-直火割烹

**類型**：花蓮無菜單料理餐廳
**核心賣點**：浮誇感

## 案例摘要

Jacky 在觀察這家餐廳時，注意到它有一種難以忽視的「浮誇特質」——無論是擺盤、食材呈現或用餐氛圍，都帶著一股刻意放大的視覺張力。多數人可能把這視為風格缺點，但 Jacky 將它識別為核心賣點，並以此作為定位切入點。

主軸確立後，行銷溝通變得清晰一致，吸引到對此風格有共鳴的受眾，**營收隨之起飛**。

## SOSTAC 拆解

| 層次 | 內容 |
|------|------|
| **Situation** | 花蓮餐飲競爭激烈，無菜單料理同質性高，缺乏差異化記憶點 |
| **Objectives** | 建立鮮明品牌個性，提升慕名而來的預訂率 |
| **Strategy** | 以「浮誇感」為核心主張，讓這個特質從潛在缺點翻轉為品牌標誌 |
| **Tactics** | 視覺內容強化浮誇風格；文案語氣配合誇張、自信、有戲劇性 |
| **Action** | 對齊所有對外溝通（社群、菜單描述、訂位頁面） |
| **Control** | 追蹤預訂量、社群互動率、媒體提及 |

## 關鍵洞察

> 「浮誇感」不是問題，是身份。當你第一個替它命名，它就成了品牌資產。

- 賣點往往藏在觀察者「覺得怪、卻說不清楚為什麼」的感知裡
- 替模糊感知命名，是讓賣點被傳播的關鍵動作
- 定位一旦清晰，所有行銷動作的投報率會同步提升
