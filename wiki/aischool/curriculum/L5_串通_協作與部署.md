# L5 串通・協作與部署

**主題**：串＋通（AGENTS 的 N＋S）｜**9 課**｜對應原 Week4「串」＋ Week5「通」
**定位**：把多個 Agent 接成會持續產出的 Agent 工廠，再把成品部署、交付給客戶。後四課（L5-06~09）為進階選修。

## 課程

### L5-01 三種協作（2:07）
> 鉤子：**Pipeline / Sub Agent / Parallel**
一個客戶任務常含好幾個決策，一個 Agent 塞不下，拆成各司其職的 Agent 再協作。

### L5-02 拆 Sub Agent（1:58）
> 鉤子：**什麼時候該拆主從**
深入最常用的主從模式：什麼時候該把大 Agent 拆成主 Agent ＋ 子 Agent。

### L5-03 Agent 工廠（1:57）
> 鉤子：**双云雲端行銷部核心運作**
把 Agent 跟人組成一個會持續產出的 Agent 工廠。

### L5-04 追蹤成本權限（1:59）
> 鉤子：**工廠維運的底盤**
三件維運的事：追蹤、成本、權限——讓工廠長期穩定不出大包。

### L5-05 部署交付（2:07）
> 鉤子：**把成品交付給客戶**
整套課程主線最後一段：把成品交付給客戶，會交付才真的能服務客戶。

### L5-06 連接外部工具 MCP（1:58）｜進階選修
> 鉤子：**讓 Agent 直接連外部工具與資料源**

### L5-07 Claude Code 自動化 CLI（1:57）｜進階選修
> 鉤子：**敢開、會用、知道它能幹嘛**
詳見 [ailab/claude-code](../../ailab/tools/claude-code.md)。

### L5-08 長對話管理 CTX（2:03）｜進階
> 鉤子：**壓縮與對話延續，不斷線不忘記**

### L5-09 開工與收工（2:09）
> 鉤子：**凍結進度、無縫接續**
讓工作可長期累積的習慣，也是 Jacky 自己天天用的工作法（對應 `/開工`、`/收工` skill）。
