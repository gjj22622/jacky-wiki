# L6 迴圈・Agent 自動迴圈

**主題**：迴圈｜**7 課**｜新增章節（平台上線後加），對應 `/迴圈` skill
**核心方法論**：[SOSTAC 迴圈與逆推除錯](../../shuangyun/双云索引.md)（已入 shuangyun 域）
**定位**：串通是把 Agent 接成生產線做完交出去；迴圈多了「做完回頭對目標，沒到再跑一輪」這一個動作。

## 課程

### L6-01 為什麼要迴圈（3:19）
> 鉤子：**你學過的 SOSTAC，其實就是一個會自己跑的迴圈**
迴圈比串通多一個動作：做完之後回頭看結果，沒到目標就再跑一輪，到了才停。

### L6-02 SOSTAC 迴圈六格（2:41）
> 鉤子：**六格＝六個各司其職的 Agent**
現況收集 Agent → 目標（留給人，迴圈的圓心）→ 策略 → 戰術 → 行動 → 控管。

### L6-03 逆向工程定目標（3:11）
> 鉤子：**拆開一個成功案例，你的目標就不會空泛**
新手把目標寫成「提升知名度」這種沒辦法檢查的句子，迴圈就不知道何時該停。逆向工程把成功案例拆開，定出可檢查的目標。

### L6-04 每格變一個 Agent（2:50）
> 鉤子：**一格一決策，輸出剛好是下一格的輸入**
把六格每一格做成可獨立運作、獨立檢查的 Agent，迴圈就從一張圖變成能跑的東西。

### L6-05 監控對 O 打分（2:34）
> 鉤子：**每跑完一圈，拿結果回頭對一次目標**
會自己跑的東西沒人盯就會跑偏。裝上監控這個「煞車加儀表板」。

### L6-06 逆推除錯（2:46）
> 鉤子：**跑不到目標，就倒著沿 SOSTAC 一格一格往回查**
迴圈卡住時不慌的方法：倒著查，從最便宜的地方改起，改不動才往更根本處。

### L6-07 動手跑迴圈（3:06）｜實作
> 鉤子：**拆解、設計、執行，用一個 Skill 把迴圈真的跑一輪**

**對應 Skill**：`/迴圈`（執行 loop）
- **安裝**：skill 資料夾放到 `~/.claude/skills/迴圈/`（學員自己的 Claude Code 裝一次）
- **用法**：`/迴圈 拆 <你的目標>` → `/迴圈 設計` → `/迴圈 跑`
- **範例**：「`/迴圈 拆 每週競品週報，老闆三分鐘看懂`」→ SOSTAC 六格拆解表 → `/迴圈 設計` 產 loop 藍圖（骨架＋終止條件＋監控）→ `/迴圈 跑` 跑一輪＋評審打分；沒過就逆推除錯
- **交付物**：迴圈執行紀錄 ＋ 一份真的產出物（如競品週報），對應 AGENTS L2
