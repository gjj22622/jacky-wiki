# 還原檢查清單

> 換機照 [換電腦SOP](換電腦SOP.md) 做完後，用這張清單驗收。**全部打勾 = 轉移完成。**
> 在 Claude Code 打 `/移機 restore` 可讓 AI 幫你逐項確認。

---

## 1. 基礎工具（步驟 1-3、6）

- [ ] `git --version` 有版本（2.x）
- [ ] `node -v` 有版本（v24+）
- [ ] `npm -v` 有版本
- [ ] `claude --version` 有版本（2.x）
- [ ] `gh --version` 有版本
- [ ] `docker --version` 有版本（要跑 AI 大腦才必須）
- [ ] `python --version` 有版本（特定 skill 才必須）

## 2. wiki 與環境（步驟 2、4）

- [ ] `~/jacky-wiki` 資料夾存在、裡面有 `wiki/`
- [ ] `git -C ~/jacky-wiki status` 正常（在 git repo 內）
- [ ] `echo $JACKY_WIKI_HOME`（Win：`$env:JACKY_WIKI_HOME`）跑出 wiki 路徑
- [ ] Claude Code 已用正確帳號登入

## 3. Skill（步驟 5、7）

- [ ] `~/.claude/skills/` 底下有 16 個個人 skill 資料夾
- [ ] `/wiki` 有反應（jacky-wiki skill）
- [ ] `/ailab` 有反應
- [ ] `/移機` 有反應（本域 skill）
- [ ] `/作品` 有反應（portfolio 域 skill）
- [ ] `/plugin` 看得到 codex / zeabur / frontend-design / claude-code-setup 已啟用

## 4. MCP（步驟 8）

- [ ] 叫 Claude 讀一次 Google Drive → 能授權並讀到
- [ ] （其他用得到的 MCP 各測一次）

## 5. 金鑰與自建（步驟 9，按需）

- [ ] 要跑 AI 大腦：`.env` 已填 `ANYTHINGLLM_API_KEY` / `OPENROUTER_API_KEY`
- [ ] `~/bin/ffmpeg.exe` 已就位（要用 tts/影音 skill 才需要）
- [ ] `~/.claude/settings.json` 已還原（hooks / statusline 生效）

## 6. 功能性驗證（最後一關）

- [ ] 開 Claude Code → 開工自動顯示 todolist（SessionStart hook 生效）
- [ ] 跑一次 `/wiki` 查得到內容
- [ ] 在 wiki 改一個字 → `git pull --rebase` → commit → push 流程通
