## 三套方法論各自的核心

### SOSTAC（源頭：會想 + 會做）

| 字母 | 階段 |
|:--:|---|
| **S** | Situation（現況分析） |
| **O** | Objectives（目標） |
| **S** | Strategy（策略） |
| **T** | Tactics（戰術） |
| **A** | Action（行動計畫） |
| **C** | Control（控制與衡量） |

**用途**：行銷企劃從現況到管控的完整六階段。
**身分**：**國際引進**的方法論（被譽為世界前三大企劃方法論）。
**Jacky 的角色**：**整合者**——14 年深耕，從學員（2011 WBSA）到 TBSA 理事長（2025-11）。
**TBSA 中的位置**：教材主軸、認證考試核心。
**詳見**：[SOSTAC 方法論](SOSTAC方法論.md)（鄭沂珊主筆 TBSA 進階教材）。

## 三者整合：策略 → 表達 + 執行

### SOSTAC → SPEAKS 升級線（內容 → 表達）

> SOSTAC 的每一階段，Jacky 都升級為對應的口說操作步驟。

```
SOSTAC（想清楚）              SPEAKS（說明白）
S Situation    現況分析  ──升級──→  S Scan      場景掃描
O Objectives   行銷目標  ──升級──→  P Purpose   目的錨定
S Strategy     行銷策略  ──升級──→  E Engineer  架構設計
T Tactics      行銷戰術  ──升級──→  A Arm       戰術裝備
A Action       行動計畫  ──升級──→  K Kindle    點燃現場
C Control      行銷管理  ──升級──→  S Sharpen   持續精進
```

### SOSTAC → AGENTS 衍生線（內容 → AI 自動化執行）

> SOSTAC 的策略思維，Jacky 在 AI 時代衍生為 Agent 體系建構方法。

```
SOSTAC 階段                  AGENTS 階段（AI 化）
S Situation    現況分析  ──衍生──→  A Analyze   分析拆解
O Objectives   目標     ─┐
                         │
S Strategy     策略     ─┼─衍生──→  G Generate  生成建構
T Tactics      戰術     ─┘
                                    E Evaluate  評測修正
A Action       行動     ──衍生──→  N Network   串聯協作
C Control      管控     ──衍生──→  T Track     追蹤管理
                                    S Scale     規模交付
```

### 表達 ↔ 萃取 — SPEAKS 與 AGENTS 的互動

> SPEAKS 的 **S（Scan）** = AGENTS 的 **A（Analyze）** 的能力底層——**把隱性知識「問出來」**是 AGENTS 拆解流程的前置能力。

兩者都是 Jacky 從 SOSTAC 產出，在實務中互為輔助：
- 用 SPEAKS 訪談萃取師父知識 → 用 AGENTS 把知識做成 AI Agent
- 用 AGENTS 自動化生產內容 → 用 SPEAKS 把內容說明白

## IP 治理（方法論內核 vs 商標品牌分開）

> **方法論內核**（誰主導內容）≠ **商標品牌**（誰持有 IP）——三者各有不同治理模式：

| 方法論 | 內核主導 | 商標／著作權 | TBSA 使用權 |
|:--:|---|---|---|
| **SOSTAC®** | 國際社群（學界引進） | 國際第三方 | TBSA 引進為教學主軸（依國際授權規則）|
| **SPEAKS®** | **Jacky 主筆**（從 SOSTAC 升級） | **Zoe 公司持有**（小大人表達學院運營主體） | **無償授權**（Jacky 任 TBSA 理事長期間） |
| **AGENTS®** | **Jacky 主筆**（從 SOSTAC 衍生） | **双云行銷持有**（Jacky 創辦的公司） | **無償授權**（Jacky 任 TBSA 理事長期間） |

> **設計原則**：
> - **方法論內核**＝知識深度，由主筆者掌握（SOSTAC 國際社群、SPEAKS Jacky、AGENTS Jacky）
> - **商標品牌**＝品牌運營，由商業實體持有（SOSTAC 國際、SPEAKS Zoe 公司、AGENTS 双云行銷）
> - **TBSA 授權使用**＝認證教學業務，三套方法論全部都可用，但商標都不在 TBSA 身上
