## 當前不在用（觀望中）

| 工具 | 為什麼觀望 | 何時可能納入 |
|---|---|---|
| Cursor | Claude Code 已涵蓋 | 除非 Cursor 出獨家功能 |
| n8n | 排程需求小 | Jwood 自動化擴大時 |
| NotebookLM | 唯讀知識庫，跟 wiki 重疊 | 不太會納入 |
| 自架 LLM（Llama 等）| 雲端模型已夠用 + 成本高 | 隱私需求大增時 |
| Anthropic Files API | 工作流還沒成熟 | 大量 PDF 處理時 |
| Computer Use（CUA）| 用例不明 | 有明確自動化任務時 |
