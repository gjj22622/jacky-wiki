# 環境變數清單

> ⚠️ **本頁只記變數「名稱」與「去哪拿值」，絕不記錄值本身。** 金鑰/token 是機密，永不進 wiki（紅線）。
> 真正的值放在 `.env`（git-ignored）或部署平台的環境變數設定。

---

## 主表

| 變數名稱 | 用途 | 值去哪拿 | 設在哪 |
|---|---|---|---|
| `JACKY_WIKI_HOME` | 讓 skill 找得到 wiki repo（多機通用）| 指向本機 wiki 路徑（如 `C:\Users\gjj22\jacky-wiki`）| Shell profile（見下）|
| `ANYTHINGLLM_API_KEY` | AI 大腦倉庫（AnythingLLM）的 API 金鑰 | AnythingLLM 後台 → API Keys | `.env`（本機）/ 部署平台環境變數 |
| `OPENROUTER_API_KEY` | LLM + embedding（走 OpenRouter）| OpenRouter 帳號後台 | `.env`（本機）/ 部署平台環境變數 |
| `MEMBERS_JSON` | AI 大腦成員名單（雲端用環境變數，本機用檔案 fallback）| 由 publisher 流程產生 | 部署平台環境變數 |

> AI 大腦相關金鑰的完整治理見 `docs/deploy-zeabur.md` 與 `docs/ai-brain-phase0.md`。

---

## 設定方式

### `JACKY_WIKI_HOME`（建議一定要設）

**Windows PowerShell**（永久，寫進 profile）：
```powershell
# 開 profile：notepad $PROFILE （沒有就先 New-Item -ItemType File $PROFILE -Force）
# 加這行：
$env:JACKY_WIKI_HOME = "$env:USERPROFILE\jacky-wiki"
```

**Ubuntu / macOS / WSL2**（寫進 `~/.bashrc` 或 `~/.zshrc`）：
```bash
export JACKY_WIKI_HOME="$HOME/jacky-wiki"
```

設完**重開終端機**才生效。驗證：
- PowerShell：`echo $env:JACKY_WIKI_HOME`
- bash：`echo $JACKY_WIKI_HOME`

跑出來是 wiki 路徑 = 設好了。

### `.env`（AI 大腦金鑰，本機檔案，git-ignored）

放在 AI 大腦專案根目錄，內容格式：
```
ANYTHINGLLM_API_KEY=（你的金鑰）
OPENROUTER_API_KEY=（你的金鑰）
```
這個檔被 `.gitignore` 的 `**/.env` 規則擋住，**不會進 git**。換機時需手動把金鑰重新填入（去上表「值去哪拿」那欄取得）。

---

## 換機提醒

1. 金鑰**不在 wiki 裡**——換機時要去各服務後台「重新拿/重新產生」一次，填回 `.env` 或部署平台。
2. `JACKY_WIKI_HOME` 設好，多機 skill 才不會找錯 wiki 路徑。
3. `settings.local.json`（含本機權限白名單）也不進 git，換機後依需要重設。
