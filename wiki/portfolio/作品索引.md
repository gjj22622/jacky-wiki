# 作品域

> 🎯 **使命**：每次要示範作品時，**不用再想「放哪、網址多少、怎麼登入」**——一句 `/作品 <名稱>` 全給齊。
> 🔒 **金鑰紅線**：本域頁面只記**非機密**（資料夾·repo·網址·帳號名）；token/密碼放 gitignored 金庫（[README](README.md)）。

---

## 什麼時候記一筆

專案達到這些**里程碑**就歸檔（`/作品 add`）：
- ✅ 部署上 GitHub
- ✅ 上了最前端（Zeabur 等）
- ✅ 接上自有 DNS

## 怎麼用

```
/作品               # 顯示本索引 + 可用子指令
/作品 add           # 歸檔一個新作品（互動收集，自動偵測資料夾/repo）
/作品 <名稱>        # 查一個作品的完整示範資訊（頁面 + 金庫接起來）
/作品 list          # 列所有作品狀態總表
```
