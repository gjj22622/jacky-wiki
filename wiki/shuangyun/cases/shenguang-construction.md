# 晟光建設

**類型**：彰化建設公司
**核心賣點**：狂（建築狂人）

## 案例摘要

Jacky 觀察晟光建設老闆時，發現他對建築有一種旁人難以理解的執著與強度——那種「為什麼要做到這種程度」的勁兒，正常的行銷框架會把它磨平，讓老闆看起來「專業」「可靠」，但那樣就泯然眾人了。

Jacky 直接把這個特質命名為「狂」，以「建築狂人」作為老闆的個人定位，短影音內容全面圍繞這個主軸——**房屋銷售隨之起飛**。

## SOSTAC 拆解

| 層次 | 內容 |
|------|------|
| **Situation** | 彰化建案市場競爭，建設公司形象同質（穩健、安全、品質），難以記憶 |
| **Objectives** | 讓老闆成為具辨識度的公眾人物，帶動品牌信任與銷售詢問 |
| **Strategy** | 以老闆的「狂」作為品牌核心，打造建築狂人個人 IP |
| **Tactics** | 短影音為主力，內容呈現老闆「狂」的具體時刻（堅持細節、超規格工法、不被理解的偏執） |
| **Action** | 統一所有影片語氣與剪輯風格，強化「狂人」人設一致性 |
| **Control** | 追蹤影片完播率、案場詢問量、成交週期變化 |

## 關鍵洞察

> 「狂」不是缺點，是篩選器。它排開了不懂欣賞的買方，吸引了真正想要這種品質的客戶。

- 老闆個人特質本身就是賣點的來源，不必只看產品
- 「狂」這個詞比「堅持品質」強一百倍——它有畫面、有情緒、能傳播
- 短影音讓「狂」變得可觀看，而非只是口號
