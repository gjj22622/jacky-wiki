# 課程地圖 — 拆 → 建 → 修 → 串 → 通 → 迴圈

> 單一真相：平台 `course.json`（產生器 `scripts/build-course-json.js`，母源 `双云AI_School/01_影片課程`）。
> 共 **42 課 / 92 分 55 秒 / 7 個 Level**。每課一支影片，影片管線「一段 `#` 對一頁」真對齊。

## 設計骨架

整套課程是 **AGENTS 六字訣** 的時間軸展開，前面包了一段 L0 入門、後面接了一段 L6 迴圈：

```
L0 入門   →  L1 拆  →  L2/L3 建  →  L4 修  →  L5 串通  →  L6 迴圈
環境起手式    三工具      SOSTAC+      Skill+     協作+部署    自動迴圈
              好問題      最小Agent    殘酷測試
```

- **L1–L5** 對應双云原本的 5 週實體課（拆／建／修／串／通），重構成影片版。
- **L0** 是新增的入門（心態、科普、帳號、起手式、檔案邏輯）。
- **L6** 是新增的進階（SOSTAC 迴圈＋逆推除錯），對應 `/迴圈` skill，是平台上線後加的章節。
