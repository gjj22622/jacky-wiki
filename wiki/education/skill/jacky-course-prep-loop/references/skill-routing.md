# 專門 Skill 路由

通用備課 skill 管理決策與 Gate，不重做專門工具能力。

| 任務 | 優先能力 |
|---|---|
| 查 Jacky 方法論、課程與前例 | `jacky-wiki` |
| SOSTAC 迴圈設計與逆推 | `/迴圈` canonical skill |
| 五張表單內容 | `tbsa-forms` 與對應 TBSA 分析 skills |
| 簡報製作／檢查 | `presentations` 或 `powerpoint` |
| Word 教案、講義 | `documents` |
| 問卷資料、名單、分析表 | `spreadsheets` |
| 教師 AI 工作坊既有模板 | `jacky-teacher-workshop`，只作該課型參考 |
| 課堂練習設計 | `ai-course-practicum-designer` |
| 双云企業營運脈絡 | `shuangyun-jacky-agent` |
| 課後經驗升格成教材 | `internal-training`；原始事件先走 `ailab` |

若專門 skill 不存在或未安裝，明確指出缺口並以 Wiki SOP 執行，不假裝已呼叫。
