## 想貢獻內容怎麼做

| 動作 | 流程 |
|---|---|
| **更新已有頁面** | 直接編輯 .md → commit → push（在主要工作機 G2 mini Ubuntu） |
| **新增頁面** | 先看 [CLAUDE.md](../CLAUDE.md) 的命名規範與跨域寫作原則 → 寫 frontmatter → 加入對應域 → 更新 `wiki主索引.md` 頁數 |
| **跨機器同步** | push 前先 `git pull --rebase origin main`（避免衝突） |
| **記錄關鍵決策** | 寫進 [`wiki/log.md`](log.md) — 哪些素材入、哪些不入、判斷理由 |
