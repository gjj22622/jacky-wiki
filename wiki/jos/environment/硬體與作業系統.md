# 硬體與作業系統

> 這套 AI 作業系統實際跑在哪些機器上。換機時對照這張表，確認新機規格夠用。

---

## 機器清單

| 代號 | 角色 | OS | 用途 | 備註 |
|---|---|---|---|---|
| **Windows 主機** | 主力開發機 | Windows 11 Pro（10.0.26200）| 日常全部工作：wiki、coding、部署 | 本清單以此機為基準 |
| **G2 mini** | 副機 / 常駐 | Ubuntu | 多機協作、可常駐跑服務 | jacky-wiki 路徑 `/home/jacky/jacky-wiki` |
| **WSL2** | Windows 內的 Linux | Ubuntu(WSL2) | 需要 Linux 環境的任務 | 與 Windows 主機的 `~/.claude/` **分開**，兩邊都要裝 skill |

> 多機協作規範：push 前先 `git pull --rebase origin main`，避免 fast-forward 衝突（見 jacky-wiki/CLAUDE.md）。

---

## 新機最低要求

換電腦時，新機建議規格：

| 項目 | 最低 | 建議 | 為什麼 |
|---|---|---|---|
| RAM | 16 GB | 32 GB | Docker + AnythingLLM 倉庫 + 瀏覽器同時開 |
| 硬碟 | 256 GB SSD | 512 GB+ SSD | repo + Docker image + 模型快取 |
| 網路 | 穩定寬頻 | — | Claude Code / 雲端模型全靠網路 |
| OS | Windows 11 / Ubuntu 22.04+ / macOS 13+ | — | 三平台都支援 |

> **沒有 GPU 也沒關係**——主力模型（Opus/Sonnet/Haiku/GPT）全在雲端，本機不跑大模型。

## 各平台 jacky-wiki 預設路徑

| 平台 | 路徑 |
|---|---|
| Windows | `C:\Users\gjj22\jacky-wiki`（= `%USERPROFILE%\jacky-wiki`）|
| Ubuntu / macOS / WSL2 | `~/jacky-wiki`（= `/home/jacky/jacky-wiki`）|
| 通用（建議設環境變數）| `$JACKY_WIKI_HOME` → 見 [環境變數清單](環境變數清單.md) |
